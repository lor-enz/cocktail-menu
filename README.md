# Cocktail Menu

[![Live demo](https://img.shields.io/badge/GitHub%20Pages-live-green)](https://lor-enz.github.io/cocktail-menu/)

Elegant single-page cocktail list inspired by speakeasy menus. Everything lives in `index.html`.
GitHub Pages can host it without a build step. Fonts load from Google Fonts and the SVG glasses keep file-size tiny while still feeling high-end.
